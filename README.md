Prerequisites:    
npm, node
clone repository and cd into it

To run app:

Type the following to comand line:
npm i --production    
npm start

To run tests:    
npm i    

unit tests:

Type the following to command line:    
npm run unitTests    

browser tests:    
Prerequisites:    
geckodriver, firefox 47+ (included in PATH)    

npm start
npm run browserTests (on different window)
